# Generador de Contraseñas en Python

Pequeño programa en Python que permite generar contraseñas aleatorias personalizadas según las preferencias del usuario.

El usuario puede elegir si desea incluir:

- Letras minúsculas
- Letras mayúsculas
- Números
- Caracteres especiales
- Longitud de la contraseña (entre 1 y 24 caracteres)

---

## Requisitos

- Python 3.8 o superior

El programa solo utiliza módulos estándar:

- `random`
- `string`

No requiere instalación de dependencias externas.

---

##  Cómo ejecutar el programa

1. Clonar el repositorio o descargar el archivo `main.py`.

2. Abrir una terminal en la carpeta donde se encuentra el archivo.

3. Ejecutar:

```bash
python main.py
